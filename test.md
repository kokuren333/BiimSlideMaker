# 特徴とメリット

BiimSlideMaker は **Markdown** と **YAML** を使って、簡単に動画を作成できるツールです。

- **手軽さ**: Markdown でスライドを記述。
- **自動化**: YAML の台本から音声を自動生成 (TTS)。
- **効率化**: スライド、音声、字幕、ノート表示を同期。
